## The Business Book 🧾 (desktop apk) 
***minimal, reliable, offline, and personalized***



## Deskription

This apk will be run in desktop its goal is to store daily expenses and income of a running business.

Key Features: 

* storing, editing, deleting daily expenses to an excel file through user an easy interface.
* defining custom types of expenses and income with the transaction.
* can see and downloading the monthly expense/income reports in the form of pdf or excel file.
